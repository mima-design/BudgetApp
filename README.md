Simple budget management application.

# Requirements
## Backend
- Python 3.7
- Django 3.2.7
- Django rest framework 3.12.4
- other libs from requirements.txt

## Frontend
- NodeJs >= 14.17.6
- React >= 17.0.2
- other packages from packages.json

# Prerun backend
By default, app uses sqlite db.
Remember to run migrations and create superuser.
If running django from command, use port 8000.
